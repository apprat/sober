# Fold

折叠面板。

```html preview
<s-fold>
  <s-button slot="trigger">折叠</s-button>
  空山新雨后，天气晚来秋。 <br />
  明月松间照，清泉石上流 <br />
  竹喧归浣女，莲动下渔舟。 <br />
  随意春芳歇，王孙自可留。 <br />
</s-fold>
```

---

## 属性

| 名称   | 类型     | 默认值 | 同步 | 介绍    |
| ------ | ------- | ------ | --- | ------- |
| folded | boolean | false  | 是  | 是否折叠 |

| 名称    | 介绍   |
| ------- | ----- |
| trigger | 触发器 |
